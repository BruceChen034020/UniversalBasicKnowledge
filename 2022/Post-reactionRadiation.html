文字
反應後輻射
　　大部分的量子反應都能安靜地反應，意即反應全程不吸收或放出任何電磁波。但有些量子反應一定會在反應後吸放電磁波，無法阻止，且放出的光子難以攔截，彷彿在昭告天下這裡發生了反應。這個現象稱為反應後輻射。
　　

原理

　　反應後輻射通常是量子反應後產生的不完全粒子所引起的。不完全粒子藉由放出不完全光子達到完全狀態。

舉例

　　負beta衰變
　　Neutron → p + e- + Electron antineutrino
　　a3b3-d3-f2g → a3bc3-e3-fg2 + a-1b-c3e3f-g2 + ab-d3-f2g
　　其中 a3bc3-e3-fg2 是不完全質子，a-1b-c3e3f-g2 是不完全電子。
　　不完全質子會發生以下的反應，產生兩個不完全光子。
　　a3bc3-e3-fg2 + ab(光子) → a3b2c3-e3-fg2 + a(不完全光子)
　　a3b2c3-e3-fg2 + ab(光子) → a3b3c3-e3-fg2 + a(不完全光子)
　　同時，不完全電子也會發生以下反應，產生兩個不完全光子。
　　a-1b-c3e3f-g2 +ab → b-c3e3f-g2 + b
　　b-c3e3f-g2 + ab → ab-c3e3f-g2 + b
　　產生出的完全光子，會兩兩合併成一個完全光子。
　　a + b → ab
　　a + b → ab
