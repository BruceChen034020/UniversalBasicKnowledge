修正人類量子學
　　修正人類量子學是一門科學，由洞後宇宙暗銀河系的阿姆三星人提出，是針對人類世界的量子物理學作進一步的改良。(本文所指的量子物理學包含粒子物理學)
　　

歷史

　　在人類世界被發現後，人類的科學成就被公諸於世，其中的量子物理學讓許多星際科學家大為興奮，因為人類用非常簡化的基本粒子模型就能夠解釋宇宙的組成。量子物理學認為宇宙是由夸克、輕子和波色子組成。
　　之後阿姆三人進一步將原子數守恆定律套入人類的量子物理學，使修正人類量子學遵守原子數守恆定律，並且能夠構造暗物質的模型以及解釋反應後輻射的量子反應。

理論基礎

　　阿姆三人把人類的基本粒子視為分子，分子是由原子組成的。每個分子都有它的分子式，說明它的組成原子，每個量子反應的前後各個原子的個數會守恆。分子有可能式固態的、液態的或是氣態的，同一個分子在不同態時可能會有迥異的表現。相同的分子，氣態質量>液態質量>固態質量。相同態的分子，分子式用阿姆三語言書寫出來的長度和a+b的比值愈大者，質量愈小，無質量粒子例外。

　　任何次原子粒子都可以用分子式表示，其表示方法和依人類的量子力學表示方法為基準相加。例如：
　　質子 = u+u+d = abc2d-e2g + abc2d-e2g + ab-c-d2e-f = a3b3c3e-3f-1g2
　　Positive pion = ud-1 = abc2de-2g + abcd2e-1f = a2b2c3d3e-3fg
　　若一個粒子的a和b個數不相等，即為不完全粒子。
　　若一個粒子的a和b個數相等但不等於1，即為超完全粒子。
　　若一個粒子的c和e個數相加不等於0，即為暗物質。

普通反應

　　當一個量子反應發生時，反應前後原子數會守恆，舉例來說：
　　電子湮滅反應 e- + e+ → γ+ γ
　　ab-c3e3f-g2 + abc3-e3-fg2 → ab + ab
　　反應前後a的個數均為2，b為2，其餘皆為0。
　　Positive pion 衰變 π+ → μ+ + νe
　　a2b2c3d3e-3fg → abc3-e3-fg2 + abd3f2-g
　　反應前後a=2 b=2 c=3 d=3 e=-3 f=1 g=1 遵守原子數守恆。
　　人類和沉澱人對質子衰變的2個假設在此理論中也遵守原子數守恆，理論上為可能發生的反應，但是實際上因質子半衰期太長無法證明。

反應後輻射

　　當反應物和產物的基本粒子個數不同時，就會產生不完全粒子，發生反應後輻射。
　　例如負beta衰變
　　Neutron → p + e- + Electron antineutrino
　　a3b3-d3-f2g → a3bc3-e3-fg2 + a-1b-c3e3f-g2 + ab-d3-f2g
　　其中 a3bc3-e3-fg2 是不完全質子，a-1b-c3e3f-g2 是不完全電子。
　　不完全質子會發生以下的反應，產生兩個不完全光子。
　　a3bc3-e3-fg2 + ab(光子) → a3b2c3-e3-fg2 + a(不完全光子)
　　a3b2c3-e3-fg2 + ab(光子) → a3b3c3-e3-fg2 + a(不完全光子)
　　同時，不完全電子也會發生以下反應，產生兩個不完全光子。
　　a-1b-c3e3f-g2 +ab → b-c3e3f-g2 + b
　　b-c3e3f-g2 + ab → ab-c3e3f-g2 + b
　　產生出的完全光子，會兩兩合併成一個完全光子。
　　a + b → ab
　　a + b → ab

超完全粒子

　　若一個粒子的a和b個數相等但不等於1，即為超完全粒子。超完全粒子大致上相當於人類量子學的「激發態粒子」。超完全粒子最常見的形成反應是光子激發基態電子產生激發態電子。
　　ab-c3e3f-g2 + ab → a2b2-c3e3f-g2 (超完全電子)
　　超完全粒子和不完全粒子一樣是不穩定狀態，皆會傾向轉為完全粒子。超完全粒子對應到人類量子學的地方是，所有基態粒子皆為完全粒子，所有激發態粒子皆為超完全粒子，故超完全粒子和激發態粒子的定義和區分是相同的。但不一樣的地方是，人類認為激發態粒子可以用任何方法回到基態，如氫原子中，一個電子可以從n=1耀遷到n=5，要回到基態時可以先經過n=2再到n=1，或先經過n=3再到n=1，或直接一次回到n=1。以上三個例子電子經過的路徑分別為：1→5→2→1，1→5→3→1，1→5→1。但在阿姆三人的觀念中，電子會記憶自己從基態開始走過的路徑，並沿原路徑回到基態，所以在以上三個例子中，前兩個是不合法的。此外，阿姆三人認為超完全粒子和不完全粒子都比穩定的完全粒子更容易吸收光子，因此當一個光子打到氦原子的時候，若氦有電子為超完全粒子，則打到超完全電子的機率大於打到完全電子的機率，這也與人類的量子學有所出入。兩者對於非穩定態電子的概念都沒有和任何實驗結果矛盾，且兩者均能成功預測每個原子光譜的亮紋亮度比例。
